Install Cyber Panel on EC2 Instance

Introduction

Apa itu "Cyber Panel"

CyberPanel adalah salah satu control panel hosting terbaik untuk melakukan manajemen VPS. Control panel hosting ini bersifat opensources, menggunakan web server OpenLiteSpeed yang gratis dan tampilan antarmuka yang user friendly. Untuk asal usulnya CyberPanel dikenal di kalangan webmaster pada tahun 2018 sebagai alternatif cPanel.

Keunggulan CyberPanel

  1. Interface manager yang User Friendly
  2. Ringan dan cepat karena dukungan OpenLiteSpeed (gratis) dan LiteSpeed Enterprise (berbayar).
  3. Memiliki fitur Backup-Restore.
  4. Support Multi PHP.
  5. Mendukung Apache .htaccess
  6. Tersedia versi gratis dan berbayar (Enterprise).
  7. Proses instalasi yang mudah.
  8. Interface WebMail yang menarik karena menggunakan Rainloop.

Kekurangan CyberPanel

  1. Username di web panel berbeda dengan username Linux.
    Kekurangan ini akan menjadi masalah ketika terjadi kesalah pada databases, jika tiba-tiba MySQL macet maka CyberPanel tidak dapat digunakan. Masalah ini bisa di selesaikan dengan masuk sebagai root.
  2. DNS Zone tidak sepenuhnya otomatis.
    Saat ini zona DNS CyberPanel tidak sepenuhnya otomatis saat kami menambahkan situs. Ada beberapa entri rentan untuk zona DNS ini yang harus diperhatikan saat menambahkan server nama.
  3. Perlu penyesuaian untuk user baru terutama untuk yang sudah terbiasa menggunakan cPanel atau Plesk.

Configuration AWS

Create Elastic IP

IP Elastic adalah alamat IP Public milik Amazon Web Service yang berguna agar semua IP Public dari resource kita tidak akan berubah-rubah dengan seiring berjalannya waktu

Network & Security> Elastic IPs lalu klik "Allocate Elastic IP Address"

Create EC2

Amazon Elastic Compute Cloud (Amazon EC2) adalah layanan web yang menyediakan kapasitas komputasi yang aman dan dapat diubah ukurannya di cloud. Amazon EC2 menawarkan banyak opsi yang membantu Anda membangun dan menjalankan hampir semua aplikasi. Dengan kemungkinan ini, EC2 dapat dimulai dengan cepat dan mudah.

Configuration EC2

SSH to EC2 Instance

Open CMD or GitBash

cd Downloads
ssh -i pem-rizwan.pem ubuntu@<ip-addr-pub-ec2>

Update Repository

sudo -i
apt update && apt upgrade -y
