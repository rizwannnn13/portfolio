Pengertian

Cracklib adalah library untuk memeriksa apakah kata sandi mudah dipecahkan atau tidak. Ia memeriksa apakah kata sandi tidak didasarkan pada pola karakter sederhana atau kata kamus. Tujuan utama dari paket ini adalah untuk menghentikan pengguna memilih kata sandi yang mudah ditebak/diretas. Ia mencoba menghasilkan kata-kata dari kolom nama pengguna dan gecos (entri di file /etc/passwd) dan memeriksa kata sandinya.

Perhatikan bahwa cracklib bukanlah pengganti program kata sandi, dan harus digunakan bersama dengan program kata sandi yang sudah ada. Instal jika Anda ingin memastikan bahwa kata sandi setidaknya aman secara minimal.

Flash Disk

Jika kita ingin mengaktifkan atau agar flash disk bisa terbaca oleh sistem linux, kita harus mengetik command

mount /dev/sdb1 /mnt

Coba kita cek apakah flash disk sudah bisa terbaca oleh sistem linux atau belum. Cara untuk kita cek adalah masuk ke direktory yang telah kita tentukan tadi yaitu /mnt

cd /mnt

ls

Buat file no-usb menggunakan nano pada direktori modprob

echo ‘install usb-storage /bin/true’ >> /etc/modprobe.d/disable-usb-storage.conf

Jika kita ingin menonaktifkan flash disk bisa mengetik command

umount /dev/sdb1 /mnt

Maka hasilnya jika kita masuk ke directory /mnt maka semua isi file yang ada di flash disk akan menghilang

Blok Ping

Untuk blok ping dari luar bisa mengetik command

echo "1" > /proc/sys/net/ipv4/icmp_echo_ignore_all

Jika ingin mengembalikan lagi agar bisa ping ke luar bisa dengan menggunakan command

echo "0" > /proc/sys/net/ipv4/icmp_echo_ignore_all

Jika ingin blok ping secara permanen dari luar maka bisa ganti isi file dibawah ini:

nano /etc/ sysctl.conf

tambahkan baris
net.ipv4.icmp_echo_ignore_all = 1

Lalu reload konfigurasi terlebih dahulu agar bisa berjalan dengan baik dan benar

sysctl -p

Strong Password

Install Package

apt-get install libpam-cracklib

Perintah untuk mengetahui informasi tanggal kadaluarsa suatu kata sandi

chage (user)

chage rizwan2
chage -l rizwan2

Meminta user untuk menggunakan kata sandi yang cukup aman dengan kombinasi yang rumit

nano /etc/pam.d/system-auth

tambahkan
/lib/security/$ISA/pam_cracklib.so retry=3 minlen=8 lcredit=-1 ucredit=-1 dcredit=-1 ocredit=-1

Lalu kita cek apakah konfigurasi tadi sudah berhasil atau belum. Cara cek bisa dengan ganti password user dengan katasandi yang mudah.

passwd (user)

passwd rizwan2
password : rizwan2

Lalu kita akan konfigurasi software libpam-cracklib agar saat mengganti password user tidak bisa dengan menggunakan katasandi yang mudah. Hal ini tentu saja akan meningkatkan kualitas security kita agar tidak mudah diketahui oleh orang lain.

nano /etc/pam.d/common-password

pada text
password requisite pam_cracklib.so retry=3 minlen=8 difok=3

ditambahkan menjadi
password requisite pam_cracklib.so retry=3 minlen=9 dcredit=-1 ucredit=-1 lcredit=-1 ocredit=-1 difok=3

Lalu kita cek apakah konfigurasi tadi sudah berhasil atau belum. Cara cek bisa dengan ganti password user dengan katasandi yang mudah. Jika saat kita mengganti password dengan menggunakan katasandi yang mudah ditolak itu artinya konfigurasi telah berhasil. Maka yang kita harus lakukan adalah mengganti password dengan menggunakan katasandi yang lebih rumit.

passwd (user)

passwd rizwan2
password : rizwan2
(ditolak)
password : R1zw4n2@123
