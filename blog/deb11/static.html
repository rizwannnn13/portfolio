Github

Buka browser dan masuk ke github untuk mencari link repository yang akan kita clone kedalam server local. Sebagai contoh saya akan menggunakan link repository dari github saya sendiri dan akan dimasukan kedalam directory static. Mari kita lihat bagaimana cara clone repository github

git clone https://github.com/rizwannnn13/portfolio.git /var/www/static

Restart Apache2

systemctl restart apache2
