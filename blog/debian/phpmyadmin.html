Install Package

PHP

apt install php php-cgi php-mysqli php-pear php-mbstring php-gettext libapache2-mod-php php-common php-phpseclib php-mysql -y

Untuk mengecek versi php kita bisa menggunakan command

php --version

MariaDB

apt install mariadb-server mariadb-client -y

Konfigurasi Database

Selanjutnya kita lakukan konfigurasi awal database MariaDB menggunakan perintah berikut:

mysql_secure_installation

(Set password root : 123 = agar mudah di ingat)

Enter current password for root (enter for none): 123

Set root password? [Y/n] y
New password: 123
Re-enter new password: 123

Remove anonymous users? [Y/n] y

Disallow root login remotely? [Y/n] y

Remove test database and access to it? [Y/n] y

Reload privilege tables now? [Y/n] y

Membuat Database

Kali ini kita akan membuat database baru di MariaDB dengan menggunakan command

mysql -u root -p

Contoh : saya akan membuat database dengan nama "db_pemula" dan user "rizwanpemula" dengan password "123". Maka ini command yang harus dijalankan. 

CREATE DATABASE db_pemula;

CREATE USER 'rizwanpemula'@'localhost' IDENTIFIED BY '123';
    
GRANT ALL PRIVILEGES ON *.* TO 'rizwanpemula'@'localhost';
    
FLUSH PRIVILEGES;
    
EXIT;

Install gnupg

apt-get -y install gnupg

Membuat Directory

Kita akan membuat direktori untuk menyimpan file phpmyadmin

mkdir /var/www/phpmyadmin

(lalu masuk ke direktori yang telah dibuat)

cd /var/www/phpmyadmin

Install paket installer phpMyAdmin

wget https://www.phpmyadmin.net/downloads/phpMyAdmin-latest-all-languages.tar.gz --no-check-certificate

wget https://www.phpmyadmin.net/downloads/phpMyAdmin-latest-all-languages.tar.gz.asc --no-check-certificate

Cek digital signature file installernya menggunakan gpg/gnupg

wget https://files.phpmyadmin.net/phpmyadmin.keyring --no-check-certificate

gpg --import phpmyadmin.keyring

Unpack file installer konfigurasi phpMyAdmin

tar xvf phpMyAdmin-latest-all-languages.tar.gz --strip-components=1 -C /var/www/phpmyadmin

Konfigurasi virtualhost untuk PhpMyAdmin

Buat dan Konfigurasi file virtualhost phpmyadmin.conf

nano /etc/apache2/sites-available/phpmyadmin.conf

<VirtualHost *:80>
    ServerAdmin rizwan@rizwanpemula.com
    ServerName phpmyadmin.rizwanpemula.com
    DocumentRoot /var/www/phpmyadmin
    ErrorLog ${APACHE_LOG_DIR}/error.log
    CustomLog ${APACHE_LOG_DIR}/access.log combined
</VirtualHost>

Aktifkan virtualhost phpmyadmin.conf

a2ensite phpmyadmin.conf

systemctl reload apache2

Konfigurasi Selesai

Untuk pengetesan gunakan domain yang kalian pakai. Disini saya menggunakan sub domain phpmyadmin.rizwanpemula.com

Login menggunakan user dan password yang kita sudah buat di mysql
