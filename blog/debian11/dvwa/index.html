Penjelasan

Singkatan dari Damn Vulnerable Web App adalah sebuah aplikasi web PHP/MySQL yang sangan rentan terhadap berbagai serangan keamanan siber. DVWA digunakan untuk pelatihan web, pentester, dan profesional Cyber Security. Dengan gunakan DVWA sebagai alat belajar athical hacker, penetration tester atau Cyber Security Analyst yang dapat mempraktikkan dan tingkatkan kemampuan dalam mengatasi berbagai masalah kerentanan keamanan pada aplikasi web.

Install Package

apt install apache2 mariadb-server mariadb-client php php-mysqli php-gd libapache2-mod-php docker -y

Setup mysql

mysql_secure_installation adalah script shell yang terdapat dalam sistem Unix dan memungkinkan provers dalam mengembangkan keamanan MariaDB yang dimiliki dengan beberapa cara seperti: Provers dapat membuat password untuk akun root. Provers dapat menghapus akun root yang dapat diakses dari luar local host.

mysql_secure_installation
#tekan 'y' untuk semua pilihan#

Database & User

mysql -u root -p

CREATE DATABASE dvwa_rizwan;
CREATE USER 'admin'@'127.0.0.1' IDENTIFIED BY '12345678';
GRANT ALL PRIVILEGES ON dvwa_rizwan.* TO 'admin'@'127.0.0.1';
FLUSH PRIVILEGES;
EXIT;

Install DVWA

git clone https://github.com/digininja/DVWA.git /var/www/dvwa

Ubah Permission

chmod 777 /var/www/dvwa/

Copy File

cd /var/www/dvwa/config
cp config.inc.php.dist config.inc.php

Ubah File

nano config.inc.php

$_DVWA = array();
$_DVWA[ 'db_server' ]   = getenv('DB_SERVER') ?: '127.0.0.1';
$_DVWA[ 'db_database' ] = 'dvwa_rizwan';
$_DVWA[ 'db_user' ]     = 'admin';
$_DVWA[ 'db_password' ] = '12345678';
$_DVWA[ 'db_port']      = '3306';

Restart Service

systemctl restart apache2 mariadb mysql
